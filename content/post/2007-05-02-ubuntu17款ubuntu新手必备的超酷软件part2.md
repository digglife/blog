继续&#8230;..

如果你还没有尝试过Linux,最新的<a href="https://www.digglife.net/archives/54922" target="_blank">Ubuntu 7.04 Feisty Fawn</a> 给你提供了一个很好的机会去尝试开源软件的乐趣.很多资深极客们都认为Ubuntu 7.04是有史以来最好最简单的Linux发行版,最重要的是它给厌倦了Windows的用户提供了一个简单安全的渠道去体验Linux. 

但是Ubuntu同样也有一些Linux老手才能理解的黑色区域.下面的这些软件能够让Ubuntu新手们的过渡更加舒适和平稳.尽管Ubuntu已经提供了普通用户需要的软件(比如Firefox,Open Office,Rhythmbox等),但是,如果你不习惯Firefox,或者需要更多的音视频播放器怎么办.那你就需要自己下载一些更加合适的软件了.虽然Ubuntu下的好软件很多很多,但是,下面的这些免费软件比起其他选择来说更加棒,更能增强新手的开源体验. 

&nbsp;

&nbsp;

### 11.<a href="http://sourceforge.net/projects/gnomebaker" target="_blank">Gnomebaker</a>

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6gnomebaker-shot31.png" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="343" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6gnomebaker-shot-thumb12.png" width="450" border="0" /></a> 

Gnome下的光盘烧录工具.

下载安装:

> sudo apt-get install k3d&nbsp; 
> 
> 或 sudo apt-get install gnomebaker

&nbsp;

### 12.<a href="http://kmymoney2.sourceforge.net/index-home.html" target="_blank">KMyMoney 0.8.6</a>

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6kmymoney31.jpg" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="343" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6kmymoney-thumb11.jpg" width="450" border="0" /></a> 

个人财务管理工具.上手简单,非常适合没有任何技术背景的用户.

<a href="https://sourceforge.net/project/showfiles.php?group_id=4708" target="_blank">下载页面</a>

&nbsp;

&nbsp;

&nbsp;

### 13.<a href="http://www.mythtv.org/" target="_blank">MythTV</a>

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6mythtv31.png" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="337" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6mythtv-thumb11.png" width="450" border="0" /></a> 

网络电视.这个工具让你可以将电脑变成一个功能齐全的媒体中心,它支持暂停,快进,回放以及录制功能.

<a href="http://www.mythtv.org/modules.php?name=Downloads&d_op=viewdownload&cid=1" target="_blank">下载页面</a>

&nbsp;

### 14.[Opera 9][1]

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6opera31.jpg" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="365" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6opera-thumb11.jpg" width="450" border="0" /></a> 

虽然Ubuntu捆绑了Firefox 2.0,但是相信很多人喜欢Opera飞快的速度和特色功能.你可以直接从<a href="http://www.opera.com" target="_blank">官方网站</a>下载.deb安装包安装.然后会在程序&#8211;internet里面看到.

&nbsp;

### 15.<a href="http://scribes.sourceforge.net/" target="_blank">Scribe</a>

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6capturescribe51.png" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="390" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6capturescribe-thumb31.png" width="450" border="0" /></a> 

文本编辑工具.Scribe是一个Gnome桌面下的文本编辑工具.它提供模板,标签式编辑和浏览,标记行数等功能.这里有<a href="http://scribes.sourceforge.net/demo.htm" target="_blank">Scribe的演示Flash</a>,看过之后就明白了.

<a href="http://downloads.sourceforge.net/scribes/scribes-0.3.2.2.tar.bz2?use_mirror=internap" target="_blank">下载页面</a>

&nbsp;

### 16.[Thunderbird 2.0][2]

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6thunderbird3.png" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="354" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6thunderbird-thumb11.png" width="450" border="0" /></a> 

鼎鼎有名的邮件客户端工具.和Ubuntu的Evolution相比一个最大的优点就是支持众多插件,比如ScribeFire博客编辑工具等等.直接到<a href="http://www.mozilla.com/en-US/thunderbird/all.html" target="_blank">官方网站</a>下载,不用安装直接运行.

&nbsp;

&nbsp;

17.<a href="http://www.videolan.org/vlc/" target="_blank">VLC 0.8.6b</a>

<a href="https://www.digglife.net/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6vlcmac3.jpg" atomicselection="true"><img style="border-top-width: 0px; border-left-width: 0px; border-bottom-width: 0px; border-right-width: 0px" height="337" src="http://digglife.qiniudn.com/wp-content/uploads/3/379/2007/05/windowslivewriterubuntu17ubuntupart.2-6b6vlcmac-thumb1.jpg" width="450" border="0" /></a> 

VLC被誉为最佳的跨平台媒体播放器.平均每秒钟1.3次下载的记录也的确证明了它的确很流行.

安装方法(确保你的源已经添加):

> sudo apt-get update
> 
> sudo apt-get install vlc vlc-plugin-esd

当然你也可以自行下载安装. 

&nbsp; 

&nbsp; 

其实在你安装了一些不错的软件之后,你会发现在Ubuntu使用起来完全不输给Windows,也许你还会觉得Ubuntu更加棒.
